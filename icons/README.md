# PWA 앱 아이콘 생성 가이드

## 아이콘 생성 방법

1. `generate-icons.html` 파일을 브라우저에서 엽니다
2. 각 크기별로 "다운로드" 버튼을 클릭합니다
3. 다운로드된 파일을 `icons/` 폴더에 저장합니다

## 필요한 아이콘 크기

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## 또는 온라인 도구 사용

다음 사이트에서 아이콘을 생성할 수 있습니다:
- https://www.pwabuilder.com/imageGenerator
- https://realfavicongenerator.net/

512x512 크기의 기본 이미지를 업로드하면 모든 크기가 자동 생성됩니다.
